# Iris PCA Feature Reduction & Visualization 🌸

An interactive Machine Learning platform for high-dimensional data analysis, featuring both exploratory forensics and a premium Streamlit dashboard.

## 🚀 Overview
This project demonstrates the power of **Principal Component Analysis (PCA)** in reducing complex, multi-dimensional datasets (like the Iris dataset) into intuitive 2D and 3D visualizations while retaining over **99% of the information**.

## ✨ Key Features
- **Exploratory Analysis**: Automated pairplot generation and feature correlation maps.
- **Dimensionality Reduction**: Step-by-step PCA transformation from 4 original features to 2 or 3 Principal Components.
- **Interactive Dashboard**: A full-featured Streamlit application with:
    - Real-time PCA parameter tuning.
    - Interactive 3D point cloud exploration.
    - Deep-dive feature loading analysis.
- **Educational Component**: Clear metrics on explained variance and feature importance.

## 🛠️ Technology Stack
- **Languages**: Python
- **Libraries**: Scikit-Learn, Pandas, Numpy, Plotly, Seaborn, Matplotlib
- **Framework**: Streamlit (Front-end)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/shridharsingh-46/PCA-Feature-Reduction-with-Visualization.git
   cd PCA-Feature-Reduction-with-Visualization
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch the Explorer**:
   ```bash
   streamlit run iris_app.py
   ```

- **Total Variance Retained (2 Components)**: 95.8%
- **Total Variance Retained (3 Components)**: 99.5%
- **Species Separation**: Clear clustering achieved between Setosa, Versicolor, and Virginica species.
